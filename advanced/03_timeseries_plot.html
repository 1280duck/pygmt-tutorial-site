進階 3：時間序列折線圖

用 PyGMT 畫簡單的時間序列圖（非地圖座標）

回首頁 觀念整理 程式碼 練習

1. 觀念整理

雖然 PyGMT 主要拿來畫地圖，但它也能畫一般的 2D 圖表， 例如時間序列折線圖。

這裡我們用假資料模擬 12 個月的某個指數（例如 NINO3.4）， 示範如何設定 X/Y 軸、標題與座標刻度。

2. 範例程式碼

存成 examples/06_timeseries.py：

import numpy as np
import pygmt

def main():
    fig = pygmt.Figure()

    # 模擬 12 個月的資料
    months = np.arange(1, 13)
    values = np.array([0.2, 0.4, 0.8, 1.1, 1.3, 1.0, 0.6, 0.1, -0.2, -0.4, -0.3, 0.0])

    # 建立一個 2D 座標系
    fig.basemap(
        region=[0.5, 12.5, -1.5, 1.5],
        projection="X12c/6c",
        frame=["WSne", "xafg+tMonth", "yafg+tIndex"],
    )

    # 畫折線與點
    fig.plot(
        x=months,
        y=values,
        pen="1.5p,blue",
    )
    fig.plot(
        x=months,
        y=values,
        style="c0.2c",
        fill="blue",
    )

    fig.show()
    # fig.savefig("timeseries_example.png")

if __name__ == "__main__":
    main()

3. 小練習

  1. 把資料換成你自己的實際時間序列（例如 NINO3.4、降水量等）。
  2. 嘗試用不同顏色或線型區分多條曲線。
  3. 修改 region 上下界，使 Y 軸範圍更貼近資料。
