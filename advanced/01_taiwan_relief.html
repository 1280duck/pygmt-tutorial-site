進階 1：台灣地形陰影圖

使用較高解析度地形資料，搭配陰影與海岸線

回首頁 觀念整理 程式碼 練習

1. 觀念整理

這一節聚焦在台灣附近區域，使用 load_earth_relief() 載入較高解析度的地形， 再用 grdimage() 搭配陰影效果顯示山脈起伏。

重要參數：

  • resolution="15s"：約 15 秒弧度解析度（較細緻，但檔案較大）。
  • region=[118, 123.5, 20, 26]：經緯度範圍，大致涵蓋台灣與周邊海域。
  • shading=True 或 intensity：加入地形陰影。

2. 範例程式碼

存成 examples/04_taiwan_relief.py：

import pygmt
from pygmt.datasets import load_earth_relief

def main():
    fig = pygmt.Figure()

    # 台灣附近區域：經緯度 [118E, 123.5E, 20N, 26N]
    region = [118, 123.5, 20, 26]

    # 載入較高解析度地形（15 秒），並裁切到指定區域
    grid = load_earth_relief(
        resolution="15s",
        region=region,
        registration="gridline",
    )

    # 畫出台灣地形陰影圖
    fig.grdimage(
        grid=grid,
        region=region,
        projection="M12c",
        frame=["af", "+tTaiwan relief"],
        cmap="geo",
        shading=True,
    )

    # 疊上海岸線與國界
    fig.coast(
        region=region,
        projection="M12c",
        shorelines="1p,black",
        borders="1/0.5p,black",
    )

    fig.colorbar(frame=["a1000", "+L高度/深度 (m)"])

    fig.show()
    # fig.savefig("taiwan_relief.png")

if __name__ == "__main__":
    main()

3. 小練習

  1. 把 region 改成你感興趣的區域，例如日本或南美洲。
  2. 調整解析度（例如改為 "30s" 或 "01m"），比較圖像細緻程度。
  3. 試著把 cmap 改成 "relief" 或其他喜歡的配色。
