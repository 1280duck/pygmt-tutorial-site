進階 2：多投影子圖

在同一張圖中排版多種世界地圖投影

回首頁 觀念整理 程式碼 練習

1. 觀念整理

PyGMT 的 subplot 介面可以一次建立多個子圖， 每個子圖可以有不同投影或不同區域，非常適合比較不同投影的地圖。

這個範例會做出 2x2 的世界地圖：

  • 上左：Mercator
  • 上右：Robinson
  • 下左：Hammer
  • 下右：Mollweide

2. 範例程式碼

存成 examples/05_subplots_projections.py：

import pygmt

def main():
    fig = pygmt.Figure()

    # 建立 2x2 的子圖框架
    fig.subplot(
        nrows=2,
        ncols=2,
        figsize=("18c", "12c"),
        frame="af",
        margins=["0.2c", "0.4c"],
        sharex="b",
        sharey="l",
        autolabel=True,
    )

    # 子圖 1：Mercator
    fig.subplot_set(panel=0)
    fig.coast(
        region="g",
        projection="M8c",
        frame=["af", "+tMercator"],
        land="lightgray",
        water="white",
        shorelines="0.5p,black",
    )

    # 子圖 2：Robinson
    fig.subplot_set(panel=1)
    fig.coast(
        region="g",
        projection="R8c",
        frame=["af", "+tRobinson"],
        land="lightgray",
        water="white",
        shorelines="0.5p,black",
    )

    # 子圖 3：Hammer
    fig.subplot_set(panel=2)
    fig.coast(
        region="g",
        projection="H0/8c",
        frame=["af", "+tHammer"],
        land="lightgray",
        water="white",
        shorelines="0.5p,black",
    )

    # 子圖 4：Mollweide
    fig.subplot_set(panel=3)
    fig.coast(
        region="g",
        projection="W0/8c",
        frame=["af", "+tMollweide"],
        land="lightgray",
        water="white",
        shorelines="0.5p,black",
    )

    fig.subplot_end()

    fig.show()
    # fig.savefig("projection_subplots.png")

if __name__ == "__main__":
    main()

3. 小練習

  1. 把其中一個子圖改成只顯示某個區域（例如西太平洋）。
  2. 改變 figsize 或子圖間距，調整版面配置。
  3. 在某個子圖上加上簡單的標註（例如經緯度格線更密、改顏色）。
