PyGMT 教學筆記（基礎篇）

用 Python 玩 GMT：從安裝到畫出第一張地圖

PyGMT 是什麼？ 安裝教學 基礎章節 參考資料

1. PyGMT 與 GMT 簡介

GMT（Generic Mapping Tools）是一套專門用於科學繪圖與地圖製作的開源工具， 由 Pål Wessel 與 Walter H. F. Smith 發展，廣泛應用在地球物理與海洋科學。

PyGMT 是 GMT 的 Python 介面，讓你可以：

  • 在 Python 中直接呼叫 GMT 的繪圖與資料處理功能。
  • 與 NumPy、Pandas、Xarray 等套件整合，做資料分析＋出地圖。
  • 在 Jupyter / JupyterLab 裡互動式畫圖，適合教學與報告。

2. 安裝 GMT / PyGMT（使用 conda）

2.1 安裝 Anaconda 或 Miniconda

建議先安裝 Anaconda 或 Miniconda，方便使用 conda 來管理環境。

2.2 設定 conda-forge 頻道

在終端機（Linux / macOS）或 Anaconda Prompt（Windows）輸入：

conda config --prepend channels conda-forge

2.3 建立環境並安裝 GMT / PyGMT

只練 GMT 指令列：

conda create --name mygmt python=3.9 gmt

若要直接用 PyGMT（會連 GMT 一起裝好）：

conda create --name mygmt python=3.9 numpy pandas xarray netcdf4 packaging pygmt

2.4 啟動環境並測試

conda activate mygmt
gmt help

測試 PyGMT：

import pygmt
pygmt.show_versions()

2.5（選用）安裝 JupyterLab

conda install jupyterlab
jupyter lab

3. 基礎章節總覽

建議依序閱讀：

  1. 基礎 1：第一張世界地圖（coast + 投影）
  2. 基礎 2：區域地圖（指定經緯度範圍）
  3. 基礎 3：地形與海深色階圖（grdimage + colorbar）

對應的 Python 程式碼都放在 examples/ 資料夾，實際跑一次會更有感覺。
