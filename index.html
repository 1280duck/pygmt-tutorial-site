2. 互動網站概念

  • HTML：決定頁面有哪些區塊、文字、按鈕。
  • CSS：決定顏色、排版、字型。
  • JavaScript：決定「互動行為」（例如按鈕被點擊後要做什麼）。

把這三個東西組合起來，再放到 GitHub 上， 開啟 GitHub Pages，就能得到一個隨時可連結分享的互動網站。
