基礎 1：第一張世界地圖

學會使用 Figure.coast() 與簡單投影

回首頁 觀念整理 程式碼 練習

1. 觀念整理

這一節用最簡單的 Hammer 投影世界地圖，認識 Figure 和 coast() 的基本參數。

  • region="g"：global，整個地球。
  • projection="H0/15c"：Hammer 投影，中心經度 0，圖寬 15 cm。
  • frame="afg"：自動刻度（a）、框線（f）、網格線（g）。
  • land、water：陸地與海洋填色。

2. 範例程式碼

存成 examples/01_first_map.py 再執行：

import pygmt

def main():
    fig = pygmt.Figure()

    fig.coast(
        region="g",
        projection="H0/15c",
        frame="afg",
        land="lightgray",
        water="white",
        shorelines="1p,black",
    )

    fig.show()
    # 若要輸出檔案，可改成：
    # fig.savefig("first_world_map.png")

if __name__ == "__main__":
    main()

3. 小練習

  1. 把投影改成 "W0/15c"（Winkel Tripel）比較差異。
  2. 把 land 改成其他顏色，例如 "lightgreen"。
  3. 把圖寬改成 "H0/20c"，觀察輸出大小變化。
