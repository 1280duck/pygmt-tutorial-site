基礎 3：地形與海深色階圖

載入全球地形網格，使用 grdimage 與 colorbar

回首頁 觀念整理 程式碼 練習

1. 觀念整理

PyGMT 內建 load_earth_relief() 可以載入地形/海深網格， 再用 grdimage() 畫成色階圖，搭配 colorbar() 顯示高度刻度。

2. 範例程式碼

存成 examples/03_relief_map.py：

import pygmt
from pygmt.datasets import load_earth_relief

def main():
    fig = pygmt.Figure()

    # 30 分解析度全球地形/海深
    grid = load_earth_relief(resolution="30m", registration="gridline")

    fig.grdimage(
        grid=grid,
        projection="R15c",   # Robinson 投影
        frame="af",
        cmap="geo",          # 地形/海深配色
        shading=True,
    )

    fig.colorbar(
        frame=["a2000", "+L高度/深度 (m)"],
    )

    fig.show()
    # fig.savefig("earth_relief_map.png")

if __name__ == "__main__":
    main()

3. 小練習

  1. 把投影改成 "M15c" 或只畫出某個區域。
  2. 改變解析度（例如 "01d"、"05m"）比較檔案大小與細緻度。
  3. 把 cmap 改成其他色階（如 "relief"），看看差異。
