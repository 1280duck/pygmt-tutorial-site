基礎 2：畫出區域地圖（以西太平洋為例）

練習用 region 指定經緯度範圍

回首頁 觀念整理 程式碼 練習

1. 觀念整理

實際做研究時，多半只關心特定區域。PyGMT 用 region="W/E/S/N" 來指定範圍，單位為度。

例如 "110/160/0/50" 表示 110°E~160°E、0°N~50°N。

2. 範例程式碼

存成 examples/02_region_map.py：

import pygmt

def main():
    fig = pygmt.Figure()

    # 西太平洋：110E~160E, 0N~50N
    region = "110/160/0/50"

    fig.coast(
        region=region,
        projection="M15c",                 # Plate Carrée 投影
        frame=["af", "+tWest Pacific"],    # 自動刻度 + 標題
        land="lightgray",
        water="lightblue",
        shorelines="1p,black",
        borders="1/0.5p,black",            # 國界
    )

    fig.show()
    # fig.savefig("west_pacific_map.png")

if __name__ == "__main__":
    main()

3. 小練習

  1. 把 region 改成台灣附近或南美洲。
  2. 試著加入 borders="2/0.5p,red" 觀察國界差異。
  3. 把標題改成中文，例如 "+t台灣周邊海域"。
